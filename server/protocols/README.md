# server/protocols

本目录存放服务端所属的协议文档。

范围：

- 服务端作为上游服务提供方时，对客户端、控制台或外部调用方暴露的稳定契约。
- 服务端内部与持久化层、对象存储、任务快照、归一化文档之间的稳定格式。
- 服务端侧 checkpoint / 断点续传接收协议。
- 服务端向下游发布内容的 pub-sub 事件协议。

当前目录：

- `client-cli/`：服务端对客户端执行层暴露的上游协议。
- `checkpoint/`：服务端侧 checkpoint / 断点续传接收协议。
- `pubsub/`：服务端发布-订阅事件协议与持久事件总线。
- `server-web/`：服务端对控制台暴露的上游 HTTP / JSON 协议。
- `storage/`：服务端持久化协议。

协议报文、协议状态机和协议边界内执行适配放在这里。模块执行逻辑放入 `server/modules`，应用编排放入 `server/application`。
